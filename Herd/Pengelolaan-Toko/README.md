## 🧩 Deskripsi Proyek
**RavShop** adalah aplikasi pengelolaan toko sederhana berbasis **Laravel** dan **Bootstrap**.  
Website ini dibuat untuk memenuhi tugas **Ujian Tengah Semester (UTS)** mata kuliah *Pemrograman Web*.

Semua data diolah menggunakan **Controller tanpa database**, ditampilkan dengan **Blade Template**, dan menggunakan komponen Blade reusable (`x-navbar` & `x-footer`).  
Desain web dibuat modern dan berwarna dengan bantuan **Bootstrap** dan **custom CSS**.

---

## ⚙️ Fitur Utama
- ✅ Login sederhana tanpa database  
- ✅ Dashboard menampilkan nama pengguna & statistik toko  
- ✅ Halaman Pengelolaan Produk (data dari Controller dengan `@foreach`)  
- ✅ Halaman Profile (data pengguna bisa diubah tanpa database)  
- ✅ Komponen `x-navbar` dan `x-footer` digunakan di semua halaman  
- ✅ Desain modern dan responsif (Bootstrap + CSS custom)  
- ✅ Footer otomatis menempel di bagian bawah layar (sticky footer)
